# My Portfolio

A professional portfolio website showcasing my projects, skills, and experience. Built with HTML, CSS, and JavaScript.

## 📋 Features

- **Responsive Design** - Works perfectly on all devices
- **Home Section** - Engaging hero with profile and call-to-action buttons
- **My Projects** - Showcase your projects with images and descriptions
- **About Me** - Tell your story and highlight your skills
- **Contacts** - Multiple ways to get in touch (email, phone, social media)
- **Smooth Navigation** - Smooth scrolling between sections
- **Modern Design** - Clean and professional UI with gradient accents
- **Animations** - Engaging scroll animations and transitions
